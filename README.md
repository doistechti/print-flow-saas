# print-flow-saas

Workspace do SaaS para atendimento e vendas de graficas.

Base tecnica:

- `docs/sdd-frontend-backend.md`
- `docs/plano-de-implementacao-por-sprint.md`
- `docs/backlog-tecnico-em-epicos-e-historias.md`
- `docs/base-tecnica.md`

## Estrutura inicial

- `apps/frontend`: interface operacional do produto
- `apps/backend`: nucleo de dominio e integracoes
- `packages/contracts`: tipos e contratos compartilhados

## Proximo passo

Instalar dependencias e subir cada app separadamente:

- `npm install`
- `npm run dev:backend`
- `npm run dev:frontend`
